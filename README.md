# フォイル式ガス軸受 公開特許ランドスケープ

フォイル式ガス軸受（バンプ型・リーフ型等）の**公開済み特許**を俯瞰・分析した、単一HTMLの静的アプリです。

- **公開対象は公知特許の調査・分析のみ**。自社の発明アイデア・出願戦略は含みません。
- web 一次サーフェスであり網羅ではありません。出願人・年・分類は概略で、確定は原文・公報の確認を要します。

## 構成
- `index.html` … 公開ファイル（自己完結の単一HTML。データ・図を内蔵）
- `data.json` … 公知特許の客観データ（手キュレーション。唯一の正）
- `fig/` … 一般的な構造の参照図（特定特許の図面ではない）

`index.html` は内部のビルドスクリプトで `data.json` ＋参照図から生成しています（ビルド道具はこのリポジトリには含めていません）。

## 画面
1. アプローチ俯瞰 — テーマ別に「アプローチ → 代表文献 → 残るギャップ」
2. 特許インデックス — 番号・出願人・カバー範囲。テーマ／キーワードで絞り込み、番号タップで Google Patents
3. 混雑マップ — テーマ別件数・カテゴリ／出願人の密集・FI（分類）の当たり
4. 未解決の課題 — 文献が残している客観的なギャップ
