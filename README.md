# フォイル式ガス軸受 公開特許ランドスケープ

フォイル式ガス軸受（バンプ型・リーフ型等）の**公開済み特許**を俯瞰・分析した、単一HTMLの静的アプリです。

- **公開対象は公知特許の調査・分析のみ**。自社の発明アイデア・出願戦略は含みません。
- web 一次サーフェスであり網羅ではありません。出願人・年・分類は概略で、確定は原文・公報の確認を要します。

## 構成（v2.2）
- `index.html` … アプリ本体（ダッシュボード／特許検索／図鑑の3画面）
  - ダッシュボードは全グラフ連動のクロスフィルタ（出願人・課題・構造・解決手段・適用機器・期間の任意組合せ）
- `data.js` … 1,287ファミリーの書誌・タグ・要約（機械判定の一次情報）
- `figs/` … コア特許の公報代表図（特許公報＝パブリックドメイン）

## データと方法
- **データ源**: Google Patents Public Data（BigQuery 公式データセット）。件数は優先権で束ねたファミリー単位。2025年以降は公開ラグ（最長18か月）のため件数未確定。
- **分類タグ・日本語要約**: 発明の名称＋要約（判定不能分はクレーム冒頭も）をもとにした機械判定の一次情報。
- **出願人の名寄せ**: 社名変更・事業承継・翻訳ゆれによる別カウントを統合（例: 石川島播磨→IHI、KOYO SEIKO→JTEKT、UTC/Raytheon→RTX、Samsung Techwin→Hanwha PS、珠海格力の英訳ゆれ→Gree、豊田自動織機の和英ゆれ等 13種）。全リストはアプリ内「データと方法」に記載。個人名義を会社へ寄せる推測的な統合は行いません。
- **コア特許の選定**: ①法域展開・成立・ファミリー規模の客観スコア上位 ＋ ②課題・構造・年代・主要出願人のカバレッジ補完 ＋ ③主要メーカー（名寄せ後15ファミリー以上の企業）は全件。コア特許には公報代表図と請求項骨子（機械要約）を付与。

デザインは The Economist のチャート様式を参照。
